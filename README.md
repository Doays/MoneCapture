# Chzzk capture 앱 소개문

기준 버전: 1.4.31

## 한 줄 소개

CHZZK 라이브와 다시보기에서 현재·이전 장면 캡처, 최대 5분 녹화, 저지연 보정과 파일 자동 정리를 지원하는 확장 프로그램입니다.

## 짧은 소개

방송을 보다가 저장하고 싶은 장면이 생기면 화면 위 버튼이나 단축키로 바로 캡처할 수 있습니다. 놓친 장면은 `-1초`처럼 설정한 시점으로 저장하고, 현재 재생 중인 영상은 최대 5분까지 녹화할 수 있습니다. 라이브 방송에서는 저지연 보정과 5초 탐색, 통나무 표시 및 자동 수령 기능도 사용할 수 있습니다.

## 상세 소개

Chzzk capture는 CHZZK 라이브와 다시보기 화면에서 자주 쓰는 캡처 기능을 간편하게 사용할 수 있도록 만든 Chromium 확장 프로그램입니다.

지금 보고 있는 장면은 `캡처` 버튼으로 바로 저장할 수 있고, 방금 지나간 장면은 `-1초`처럼 설정한 시간만큼 이전 화면으로 저장할 수 있습니다. 캡처 단축키를 누르고 있으면 연속 캡처도 가능하며, 현재 재생 중인 영상은 최대 5분까지 녹화할 수 있습니다.

라이브 방송에서는 저지연 기능이 플레이어 상태를 확인해 재생 속도나 위치를 조절합니다. `자동`, `빠른 반응`, `안정 우선`, `직접 설정` 모드를 제공하며, 현재 상태와 플레이어 기준 지연 시간은 확장 프로그램 팝업에서 확인할 수 있습니다.

저장 폴더를 선택하면 캡처와 녹화 파일이 방송별 하위 폴더에 정리됩니다. 같은 방송이 이어지는 동안 제목이 바뀌더라도 같은 폴더를 사용하며, 폴더 직접 저장을 사용할 수 없는 환경에서는 브라우저 다운로드 방식으로 저장됩니다.

그 밖에도 5초 앞뒤 이동, 화면 버튼 위치 변경, 단축키 설정, 이전 장면 캐시 조절, 녹화 품질 선택, 화면 버튼 숨기기, 통나무 표시 및 자동 수령 기능을 제공합니다.

## 주요 기능

- 현재 재생 화면을 PNG 이미지로 저장
- 설정한 시간 전 장면을 바로 캡처
- 단축키를 누르고 있는 동안 연속 캡처
- 현재 재생 중인 영상을 최대 5분까지 녹화
- 라이브 플레이어 기준 저지연 재생 보정
- 좌우 방향키를 이용한 5초 앞뒤 이동
- 방송별 저장 폴더 자동 정리
- 화면 버튼 위치와 단축키 변경
- 통나무 보유량 표시 및 수령 가능한 보상 자동 수령

## 사용 전에 알아둘 점

저지연 기능에 표시되는 시간은 송출 서버부터 시청자 화면까지의 전체 지연 시간이 아닙니다. 브라우저 플레이어가 이동할 수 있는 최신 지점과 현재 재생 위치의 차이를 기준으로 계산합니다.

`자동`과 `빠른 반응` 모드는 저지연 상태를 유지하기 위해 플레이어의 높은 화질을 다시 선택할 수 있습니다. 화질을 직접 낮춰 사용하거나 네트워크가 불안정하다면 `안정 우선` 모드가 더 적합합니다.

녹화 파일의 해상도, 프레임, 확장자는 현재 재생 중인 영상과 브라우저의 미디어 지원 상태에 따라 달라질 수 있습니다. 녹화 품질 설정은 비트레이트와 코덱 선택 기준을 조절하며, 특정 해상도나 60fps 출력을 강제로 보장하지는 않습니다.

통나무 자동 수령은 로그인된 CHZZK 계정에서 수령 가능한 보상이 확인될 때 동작합니다. 자동 수령을 사용하지 않으려면 확장 프로그램 설정에서 해당 기능을 끌 수 있습니다.

## 안내

이 확장 프로그램은 개인이 제작한 비공식 도구이며 NAVER 또는 CHZZK의 공식 확장 프로그램이 아닙니다. 캡처와 녹화 기능은 서비스 이용약관과 콘텐츠 권리를 지키는 범위에서 사용해 주세요.


## 설치 방법

1. 브라우저에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 저장소 안의 `extension` 폴더를 선택합니다.
5. CHZZK 라이브 또는 비디오 페이지를 새로고침합니다.

## 사용 방법
[사용 설명서](manual.md)
CHZZK 페이지에서 영상 정보 오른쪽에 버튼이 표시됩니다.

- `SHOT`: 현재 영상 프레임 캡쳐
- `BACK`: 설정한 시간만큼 이전 프레임 캡쳐
- `REC`: 녹화 시작/중지
- `LAT`: 라이브 딜레이 감시
- 눈 모양 버튼: 화면 버튼 숨김

팝업에서 `설정 열기`를 누르면 기능 표시 여부, 단축키, BACK 캐시, 녹화 프리셋, 저장 폴더를 설정할 수 있습니다.

## 저장 방식

기본값은 폴더 직접 저장입니다.

처음 저장할 때 폴더를 선택하면, 선택한 폴더 아래에 방송 날짜와 제목으로 된 하위 폴더가 만들어지고 그 안에 스크린샷과 녹화 파일이 저장됩니다.

같은 방송에서는 제목이 바뀌거나 자정이 지나도 같은 폴더를 유지합니다. 다음 방송은 CHZZK 방송 세션 정보를 기준으로 새 폴더로 분리합니다.

## 녹화 제한

녹화는 정책 제한으로 5분까지만 저장됩니다.

5분에 도달하면 자동으로 녹화를 멈추고 파일을 저장합니다.

## 검증

Windows 프로젝트 루트에서 실행합니다.

```powershell
.venv\Scripts\python - <<'PY'
import json
from pathlib import Path

root = Path("extension")
manifest = json.loads((root / "manifest.json").read_text(encoding="utf-8"))
assert manifest["manifest_version"] == 3
assert manifest["name"] == "Chzzk capture"
assert (root / "content.js").exists()
assert (root / "shared.js").exists()
assert (root / "icons/icon128.png").exists()

content = (root / "content.js").read_text(encoding="utf-8")
assert "RECORD_MAX_DURATION_MS" in content

print(manifest["name"], manifest["version"])
PY
```
